"A dictorial social media app where everything controlled by the administrator."

This app is being created for the special individual who wishes to own and control their own social media app, where all content wishing to be posted by users must be approved and then posted by the administrator.

ACTIONS
This program allows the administrator to:
- Add users
- Remove users
- Edit users
- Post messages on behalf of user
- Delete messages on behalf of user
- Display user details
- Display messages (user specific or all at once)
- Can display information regarding messages posted (time & date, word or character count, #hashtags)

This program allow users to:
- Login to their account
- View their details, request their details to be edited
- Request removal of their account
- View posted messages in order of most recent.
